# JalTrack Pune — Smart Water Tanker Management

JalTrack Pune is a high-fidelity, responsive web application designed to address urban water tanker management challenges in Pune. It provides a transparent, real-time platform for residents to book tankers and for municipal authorities to monitor the city-wide fleet.

## ✨ Features

### 👤 For Residents (Gokul Society)
- **Smart Booking**: Request PMC Municipal or Private tankers with dynamic pricing based on volume.
- **Live Tracking**: Real-time GPS simulation of the tanker's journey with ETA countdown and driver details.
- **History & Analytics**: Detailed logs of past deliveries and usage charts.
- **Complaint System**: File reports for overcharging, quality, or no-shows with evidence upload.
- **Shortage Alerts**: Stay informed with critical city-wide water shortage notifications.

### 🛡️ For Authorities (ADMIN-1)
- **Fleet Dashboard**: Monitor all active tankers across Pune districts.
- **Emergency Dispatch**: Instantly send tankers to critical shortage zones.
- **Complaint Management**: View and process resident reports in real-time.
- **Daily Reports**: Export comprehensive daily logs of movements and resolutions.
- **City-wide Broadcasting**: Issue emergency shortage alerts to all residents instantly.

## 🚀 Technologies Used
- **Frontend**: Vanilla HTML5, CSS3 (Custom Design System, Glassmorphism).
- **Logic**: Vanilla JavaScript (State management, navigation, and live simulations).
- **Icons**: [Lucide Icons](https://lucide.dev/).
- **Animations**: CSS Keyframes & JS-driven UI transitions.

## 🛠️ How to View Locally
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use a local HTTP server (e.g., `python -m http.server 8081`) for the best experience with dynamic components.
